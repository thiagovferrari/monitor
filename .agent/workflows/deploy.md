# Deploy para Vercel

// turbo-all

1. Commit das alterações
```bash
git add .
git commit -m "fix: corrigir cálculo de datas para horário de Brasília"
```

2. Push para o repositório
```bash
git push origin main
```

3. Verificar deploy no Vercel
- O deploy será automático após o push
- Aguardar conclusão do build
